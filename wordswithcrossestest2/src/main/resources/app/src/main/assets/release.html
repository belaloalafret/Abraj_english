Words With Crosses v2.4.0

Changes in v2.4.0:

  • Fixed Wall Street Journal downloader, which is now a daily (Monday–Saturday) puzzle, with contest puzzles on Fridays. See the blog for more info on the contest puzzles.
  • Fixed New York Times Monthly Bonus Puzzle downloader
  • Added new downloader for the New York Times Mini Puzzle, a new daily mini 5x5 puzzle; NYT subscription required
  • Removed downloaders for Matt Gaffney's Weekly Crossword Contest and the Washington Post (syndicated by CrosSynergy), which have moved to subscription-only services. You can subscribe at mgwcc.com and csxword.com respectively.
  • Fixed date picker in puzzle download dialog, which was displaying incorrectly on devices running Android 5.0 (Lollipop) and higher
  • Fixed display of accented characters in Newsday puzzles
  • Fixed display of certain characters in puzzle titles of Universal and USA Today puzzles
  • Fixed several cursor movement bugs

It's been a while, hasn't it? I've been very busy with my job and other obligations, so I apologize for the lack of updates in a while. Thanks to everyone who's been submitting feedback—keep it coming.

What do you think of the new(ish) Material Design look of Shortyz? Is that something you'd like to see in Words With Crosses? I've been hesitant to merge that code in because the initial response was mixed, and it was extremely buggy at first as well. Let me know if you have any opinions on that.

If you enjoy using Words With Crosses, please consider rating it in the Play Store.

If you ever run into any problems or want to provide any feedback (feature suggestions, puzzle source requests, etc.), please feel free to email me at <wordswithcrosses@adamrosenfield.com>. You can also file an issue or a submit a pull request on the project page on GitHub, where the source code lives.

Welcome to Words With Crosses! The wonderful Shortyz Crosswords app has not seen any updates in quite some time, so took it upon myself to fork the project into what you see before you. That that was possible at all is thanks to Shortyz' being open-source.

Many thanks to Robert Cooper for creating Shortyz.

Words With Crosses is free, will always remain free, and contains no advertising. I don't want to give the appearance that I'm reselling anyone else's property. I don't host or copy the puzzles you download in the app in any way—you download them directly from their publicly available sources with the app. This is a side project of mine done in my spare time.

That said, if you'd like to make a donation to help further the development, please tap the "Donate" link above. If you'd prefer to donate from a different computer or device, you can also donate at http://adamrosenfield.com/donate. Also consider purchasing a subscription to one of the many great paid puzzle sources listed here.
